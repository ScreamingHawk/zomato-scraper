# Zomato Scraper

Scrape Zomato for resturant information.

## Usage

Install the dependencies

```sh
yarn install
```

Build and run the program

```sh
yarn start
```

For developement, watch the files for changes

```sh
yarn watch
```

Run the script without building

```sh
yarn run
```

Build the program without running

```sh
yarn build
```
